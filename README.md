# Harmonic Collection Cocktail Menu

A beautiful, interactive cocktail menu website featuring multiple cocktail recipes with engaging animations and games.

## Features

- Interactive cocktail cards with smooth animations
- Individual pages for each cocktail with unique designs
- Super Mario-style game for Americano cocktail
- Cocktail mixing game for Tequila Sunrise
- Custom mouse cursors and interactive backgrounds
- Responsive design for all screen sizes

## Cocktails Featured

1. **Bee's Knees** - Classic gin cocktail with honey and lemon
2. **Americano** - Italian aperitif with Campari and vermouth
3. **Negroni** - Equal parts gin, Campari, and sweet vermouth
4. **Bloody Mary** - Vodka with tomato juice and spices
5. **Ruby Cinema** - Layered dessert cocktail with raspberry and pomegranate
6. **Tequila Sunrise** - Classic layered cocktail with tequila and orange juice

## Technologies Used

- HTML5
- CSS3 (with animations and gradients)
- JavaScript (for interactivity and games)
- Canvas API (for game rendering)

## How to Upload to GitHub

### Prerequisites
1. Make sure Git is installed on your system
2. Have a GitHub account

### Steps

1. **Initialize Git repository** (if not already done):
   ```bash
   git init
   ```

2. **Add all files**:
   ```bash
   git add .
   ```

3. **Create initial commit**:
   ```bash
   git commit -m "Initial commit: Harmonic Collection cocktail website"
   ```

4. **Create a new repository on GitHub**:
   - Go to https://github.com/new
   - Create a new repository (don't initialize with README)
   - Copy the repository URL

5. **Add remote repository**:
   ```bash
   git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
   ```

6. **Push to GitHub**:
   ```bash
   git branch -M main
   git push -u origin main
   ```

### If you need to configure Git for the first time:

```bash
git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"
```

## File Structure

- `home.html` - Main landing page with cocktail cards

- `americano.html` - Americano cocktail detail page with game
- `Negroni.html` - Negroni cocktail detail page
- `bloody.html` - Bloody Mary cocktail detail page
- `ruby-cinema.html` - Ruby Cinema cocktail detail page
- `tequila-sunrise.html` - Tequila Sunrise cocktail detail page with mixing game
